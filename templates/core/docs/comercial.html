Manual do Usuário: Comercial e Vendas

Gerencie sua carteira de clientes, feche contratos e controle automaticamente as comissões de parceiros (RT).

Gestão de Clientes
O ponto de partida

Antes de lançar qualquer venda, o cliente deve estar cadastrado. O sistema permite o cadastro completo de:

  • Pessoa Física: Nome, CPF, Endereço.
  • Pessoa Jurídica: Razão Social, CNPJ, Contato Financeiro.

Dica: Mantenha o cadastro limpo. Esse nome é o que aparecerá nos relatórios financeiros e nos recibos.

Contratos e Obras

É aqui que a mágica acontece. Ao fechar uma venda, você cria um "Contrato" no sistema.

O que você define aqui?
  • Quem é o Cliente.
  • Qual é o valor total fechado (Orçamento).
  • Data da assinatura.
  • Quem indicou (Parceiro/Arquiteto).

Esse contrato se torna o "pai" de todas as parcelas que o cliente vai pagar. Assim, você sabe exatamente quanto já recebeu de cada obra.

Parceiros e Arquitetos (RT)

Gerencie quem traz negócios para a empresa.

  • Comissão Automática: Ao vincular um Arquiteto a um Contrato, o sistema já sabe que existe uma "Reserva Técnica" (RT) a ser paga.
  • Transparência: O sistema mostra exatamente quanto cada parceiro gerou de vendas para a Zirk. Isso ajuda a identificar quem são seus melhores aliados comerciais.
Painel RT (Reserva Técnica)
O controle que evita erros

Existe uma tela exclusiva chamada "Painel RT". Ela cruza dados do Comercial com o Financeiro para responder perguntas difíceis:

  • "Já recebi do cliente para poder pagar o arquiteto?"
  • "Quanto ainda devo de comissão para o parceiro X?"

Isso evita que você pague uma comissão antes de receber do cliente (fluxo de caixa negativo) ou que esqueça de pagar um parceiro importante.
